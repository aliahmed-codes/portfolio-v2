# Tailor of Signboard

## Overview

Corporate website for Tailor of Signboard, one of Oman's leading signage manufacturing companies. The site showcases their extensive product range, services, and portfolio of completed projects across the Middle East.

## Features

- **Product catalog** with category filtering
- **Project portfolio** showcasing completed installations
- **Service pages** detailing manufacturing capabilities
- **Contact forms** with location mapping
- **Multi-language support** (English/Arabic)
- **Mobile-responsive** design for all devices

## Technical Stack

- **Next.js** for static site generation
- **Bootstrap 5** for responsive grid system
- **React components** for interactive elements
- **Vercel** for fast global CDN
- **Optimized images** for fast loading in Middle East regions

## Results

- **3x increase** in online inquiries
- **50% reduction** in page load time vs. previous site
- **Mobile traffic** increased to 70% of total visits
