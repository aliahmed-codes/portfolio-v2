# HKR Architects

## Overview

HKR Architects is a world-leading international architecture and urban design practice founded in 1992, with offices in London, Dubai, Abu Dhabi, and Dublin [^1^] [^9^]. The website project involved converting a complex, award-winning design from TM Studio—one of the UK's top design agencies—into a fully functional WordPress site with pixel-perfect precision.

## The Client

HKR Architects delivers exceptional design across:
- **Landmark towers** and commercial developments
- **Residential & student housing** projects
- **Hotels & resorts** hospitality design
- **Healthcare & education** facilities
- **Masterplanning & urban design** [^8^]

With 30+ years of portfolio spanning Europe, Middle East, and Africa [^69^], the site needed to showcase their global expertise while maintaining the design integrity of TM Studio's vision.

## The Challenge

TM Studio provided a design with:
- **Complex grid layouts** with asymmetric compositions
- **Typography-heavy** pages with precise spacing
- **Custom animations** and scroll-triggered effects
- **Large-scale imagery** with overlay treatments
- **Mobile breakpoints** requiring careful adaptation

The challenge was achieving 100% visual fidelity while building on WordPress for client content management.

## The Solution

### Design-to-Code Process
- **Figma-to-HTML** conversion maintaining exact specifications
- **CSS Grid & Flexbox** for complex layouts
- **Custom typography** implementation with variable fonts
- **Image optimization** pipeline for large architectural photos
- **GSAP animations** matching design motion specs

### WordPress Integration
- **Custom theme** built from scratch (no starter themes)
- **ACF Pro** for flexible project portfolios
- **Custom post types** for projects, team, offices
- **Multi-language ready** structure (WPML compatible)
- **SEO architecture** for architecture industry keywords

### Performance Optimization
- **Lazy loading** for project galleries
- **WebP image format** with fallbacks
- **Critical CSS** inline for above-fold content
- **CDN integration** for global asset delivery
- **Core Web Vitals** optimization (90+ scores)

## Key Features

### Project Portfolio
- **Masonry grid** with hover effects
- **Filterable by sector** (Residential, Commercial, Hospitality, etc.)
- **Project detail pages** with image galleries
- **Related projects** suggestions
- **Social sharing** integration

### Practice Section
- **Office locations** with interactive maps
- **Team directory** with partner profiles
- **Awards & recognition** showcase
- **Sustainability commitment** pages
- **Career opportunities** integration

### Design Fidelity
- **Pixel-perfect spacing** matching Figma specs (±1px)
- **Color accuracy** with brand guidelines
- **Typography hierarchy** exactly as designed
- **Animation timing** synchronized with prototypes
- **Responsive behavior** true to mobile designs

## Technical Implementation

### Frontend
- **Vanilla JavaScript** (no jQuery) for modern performance
- **CSS Custom Properties** for theming
- **Intersection Observer** for scroll animations
- **Responsive images** with srcset
- **Accessibility** (WCAG 2.1 AA) compliance

### WordPress Backend
- **Custom admin dashboard** for easy content updates
- **Drag-and-drop page builder** (ACF Flexible Content)
- **Automated backups** and security hardening
- **Staging environment** for content review
- **Training documentation** for client team

## Results

- **100% design fidelity** achieved vs. TM Studio specifications
- **95+ Lighthouse scores** across all categories
- **30% faster** load times than previous HKR site
- **Zero layout shifts** (CLS 0)
- **Client autonomy** with intuitive WordPress backend

## Design Agency Collaboration

Working with TM Studio (tm-studio.co.uk) [^72^]—a London-based brand consultancy—required:
- Daily communication via Slack
- Design review sessions with pixel-level feedback
- Animation specification documentation
- Asset delivery pipeline optimization
- Quality assurance against design prototypes

## Lessons Learned

- Critical importance of design handoff documentation
- Value of building static HTML before WordPress integration
- Need for robust image handling in architecture sites
- Benefits of CSS Grid for editorial layouts
- Complexity of translating agency motion designs to web
