# Appetite Creative

## Overview

Appetite Creative is a Spain-based digital marketing agency serving Fortune 500-level clients across UK and Spain. I serve as Technical Lead managing their entire web infrastructure, overseeing 15+ active projects across multiple technologies, 3 self-hosted servers, and enterprise-level security protocols.

## The Challenge

The agency needed a reliable technical partner to:

- Manage diverse client portfolio across multiple CMS platforms
- Ensure 99.9% uptime for high-traffic brand websites
- Handle emergency security incidents and recovery
- Migrate legacy sites to modern infrastructure
- Implement multi-language solutions for international brands

## Key Projects Managed

### Main Agency Website

- **Platform**: WordPress with Avada theme
- **Responsibilities**: Full maintenance, updates, security monitoring
- **Features**: Portfolio showcase, team management, blog system
- **Traffic**: 50K+ monthly visitors

### Finca Valhalla - Accommodation Booking

- **Platform**: Custom WordPress theme
- **Features**: Complete booking system using MotoPress plugin
- **Multi-language**: English, Spanish, German support
- **Custom post types**: Properties, bookings, testimonials
- **URL**: https://dev.finca-valhalla.com/

### Old Pulteney Whisky (Security Recovery)

- **Platform**: WordPress custom theme
- **Incident**: Major hacking attack compromising site integrity
- **Recovery Actions**:
  - Complete malware removal and vulnerability patching
  - Server hardening with WAF implementation
  - Database restoration from clean backups
  - Security audit and ongoing monitoring setup
  - SSL reconfiguration and CDN optimization
- **Result**: Full recovery with enhanced security, zero downtime since

### Talc Scandal UK

- **Platform**: Next.js with Strapi CMS
- **Purpose**: Claims information site against Johnson & Johnson
- **Features**: Dynamic content updates, document library, contact forms
- **CMS**: Strapi for client content management
- **URL**: https://talcscandal.co.uk/

### Medical Nutrition Dossier (MNID)

- **Platform**: Next.js with Sanity CMS
- **Challenge**: Complex document navigation like a digital book
- **Features**:
  - Chapter-based navigation system
  - Cross-referencing between sections
  - Reference number and figure linking
  - Advanced search with filters
  - Custom Sanity schemas for medical content
- **Database**: MongoDB for flexible content structure
- **URL**: https://www.mnidossier.org/

### Influent Media

- **Platform**: Next.js
- **Status**: Completed 50% incomplete project
- **Features**: Digital platform for media agencies
- **Result**: Successfully launched and deployed

## Technical Infrastructure

### Server Management

- **3 Self-hosted servers** running Ubuntu/CentOS
- **Docker containers** for application isolation
- **Nginx reverse proxy** with load balancing
- **Automated backups** with off-site storage
- **SSL certificate management** and renewal automation

### Security Protocols

- **Web Application Firewall (WAF)** configuration
- **DDoS protection** and rate limiting
- **Regular security audits** and penetration testing
- **Malware scanning** and removal procedures
- **Access control** and 2FA implementation

### Deployment Pipeline

- **Git-based workflows** with CI/CD
- **Staging environments** for client approval
- **Automated testing** before production deploys
- **Rollback capabilities** for emergency recovery
- **Uptime monitoring** with instant alerts
