# Atelier 001

## Overview

Atelier 001 is a premium e-commerce platform for one of the UK's largest lighting manufacturers. The project involved creating a pixel-perfect custom Shopify theme and a complementary Next.js dashboard for advanced order and invoice management, serving customers across UK, US, and international markets.

## The Challenge

The client needed a sophisticated e-commerce solution that could:

- Showcase high-end lighting products with immersive visuals
- Handle complex B2B and B2C pricing structures
- Manage international shipping and multi-currency transactions
- Provide robust backend tools for inventory and order management
- Maintain brand luxury aesthetic while ensuring fast performance

## The Solution

We delivered a two-part solution:

### Custom Shopify Theme

- **Pixel-perfect design implementation** from high-fidelity Figma mockups
- **Advanced product configurators** for custom lighting options
- **Multi-currency support** with real-time exchange rates
- **B2B wholesale portal** with tiered pricing and bulk ordering
- **SEO-optimized architecture** for competitive lighting industry keywords

### Next.js Admin Dashboard

- **Invoice generation system** with PDF export and email automation
- **Order management interface** with advanced filtering and bulk actions
- **Performance analytics** tracking sales, popular products, and customer behavior
- **Inventory synchronization** between Shopify and external ERP systems
- **Multi-user role management** for client team access control

## Key Features

### Frontend Experience

- Immersive product galleries with zoom and 360° views
- Room visualization tool showing lighting in different settings
- Advanced filtering by style, material, bulb type, and dimensions
- Saved favorites and project boards for interior designers
- Quick view and compare functionality

### Backend Dashboard

- Real-time order tracking and status updates
- Automated invoice generation with custom branding
- Customer management with purchase history and preferences
- Sales reporting with exportable CSV/Excel formats
- Integration with shipping providers for label generation

## Technical Implementation

### Shopify Theme

- **Liquid templating** with modular section architecture
- **Shopify CLI** for version control and deployment
- **Custom JavaScript** for interactive product features
- **Schema settings** for client-friendly content management
- **Performance optimization** achieving 90+ Lighthouse scores

### Next.js Dashboard

- **App Router** with server-side rendering for fast loads
- **SQLite database** for lightweight, efficient data storage
- **Shopify REST & GraphQL APIs** for seamless data sync
- **NextAuth.js** for secure authentication
- **Tailwind CSS** for rapid, consistent styling

### Infrastructure

- **Shopify Plus** for enterprise-level hosting
- **Vercel** for dashboard deployment with edge network
- **Shopify Webhooks** for real-time data synchronization

## Results

- **£2M+ revenue** processed in first 6 months
- **40% increase** in average order value vs. previous platform
- **65% faster** page load times improving SEO rankings
- **50% reduction** in admin time through automated invoicing
- **4.9/5 customer satisfaction** rating on checkout experience

## Lessons Learned

This project reinforced the importance of:

- Balancing visual luxury with performance optimization
- Creating flexible admin tools that grow with client needs
- Proper API rate limiting and error handling for Shopify integrations
- Mobile-first approach even for high-end B2B customers
