# Restaurant SaaS Platform

## Overview

A comprehensive SaaS platform enabling restaurants to manage digital kiosk displays, menus, and advertisements. The system supports multiple tenants with role-based access and feature restrictions based on subscription plans.

## Key Features

### Multi-Tenant Architecture
- **Isolated data** per restaurant chain
- **Custom branding** and theming options
- **White-label capabilities** for resellers
- **Scalable infrastructure** supporting 1000+ locations

### Kiosk Management
- **Remote content updates** across all devices
- **Scheduled playlists** for different times of day
- **Ad integration** with impression tracking
- **Offline mode** with local caching
- **Device monitoring** and health checks

### Role-Based Access Control
- **Super Admin**: Platform-wide management
- **Restaurant Manager**: Location-specific controls
- **Content Editor**: Menu and promotion updates
- **Viewer**: Read-only dashboard access

### Subscription Tiers
- **Basic**: Digital menus only
- **Professional**: + Ad management and analytics
- **Enterprise**: + Multi-location, API access, priority support

## Technical Stack

- **React** with Material UI for admin dashboard
- **Node.js/Express** REST API
- **MongoDB** for flexible tenant data
- **JWT authentication** with refresh tokens
- **Socket.io** for real-time kiosk updates
- **AWS S3** for media storage

## Results

- **50+ restaurant chains** onboarded
- **10,000+ kiosk devices** managed
- **99.9% uptime** achieved
- **40% reduction** in menu update time for clients
